<Steps titleSize="h3">

<Step title="Install">

Install the [CodeQR TypeScript SDK](/sdks/typescript) using your preferred package manager:

<Tabs>
  <Tab title="npm">

    ```bash
    npm install codeqr
    ```

  </Tab>
  <Tab title="yarn">
  
  ```bash
  yarn add codeqr zod # zod is a peer dependency
  ```
  
  </Tab>
  
  <Tab title="pnpm">
  
  ```bash 
  pnpm add codeqr 
  ```
  
  </Tab>
</Tabs>

</Step>

<Step title="Initialize">

Then, initialize the CodeQR TypeScript SDK with your API key.

```typescript lib/codeqr.ts
import { CodeQR } from "codeqr";

export const codeqr = new CodeQR({
  token: process.env.CODEQR_API_KEY, // optional, defaults to CODEQR_API_KEY env variable
});
```

You can now use the `codeqr` object to interact with the CodeQR API.

```typescript
import { codeqr } from "./lib/codeqr";
```

</Step>

</Steps>
